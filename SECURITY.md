# Security Policy

0pty is a PTY daemon that can expose an interactive process over a socket, so
security reports should be handled privately first.

Supported versions:

- The current `main` branch.
- The latest published GitHub Release.

If you believe you found a vulnerability, do not include exploit details in a
public issue. Use GitHub private vulnerability reporting for this repository if
it is available. If it is not available, open a public issue that only asks for
a private reporting channel and omits technical details.

Expected acknowledgement time: 7 days.
